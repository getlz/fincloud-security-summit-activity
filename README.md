# fincloud-security-summit-activity
Overview

This repository documents my participation in the FinCloud Security Summit, an event focused on cloud security, Zero Trust frameworks, artificial intelligence risks, and cybersecurity in financial systems.

The summit emphasized that as organizations integrate AI and cloud infrastructure into financial services security and trust must be built directly into system architecture. Topics included Zero Trust security models, adversarial AI risks, and secure cloud system design.

To demonstrate what I learned from the event, I created a small deliverable that includes a

A Zero Trust architecture diagram for AI-driven financial cloud systems

A Python simulation of a Zero Trust AI security gateway

These deliverables show how security concepts discussed during the event can be translated into system design and simple implementation logic.
There were others topics that I didnt mention but I included provided PowerPoint slides from the presentations, were I was authorized to keep a copy of the slides.
